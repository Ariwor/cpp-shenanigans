## More resources (recommendations by github.com/kongr45gpen)
- [cppreference.com](https://en.cppreference.com/w/), a go-to reference for any function or feature of the language.
- [ISO C++ Core Guidelines](http://isocpp.github.io/CppCoreGuidelines/CppCoreGuidelines), with a lot of best practices and background information from the C++ committee
  - [A tour of modern C++](https://isocpp.org/tour)
- [C++ notes for professionals](https://goalkicker.com/CPlusPlusBook/): This is actually a collection of stack-overflow answers to popular questions - nothing more!
- [Programming: Principles and Practice Using C++](https://www.amazon.com/dp/0321992784) which is a great, strong and long introductory book about C++ by Stroustroup, with a lot of commentary, details and thoughts.
